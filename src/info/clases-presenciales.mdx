### Elementos necesarios

_La sala cuenta con todos estos elementos, los cuales son desinfectados diariamente. En caso de querer traer tus propios elementos, podes hacerlo.
Estos elementos NO son imprescindibles para todas las prácticas y se emplean para realizar algunas posturas_.

- Una colchoneta o algo similar para el piso.
- Una pelota o almohada.
- Una banda elástica, cinto, pañuelo largo o media cancan sin uso.
- Un palo de escoba.
- Un ladrillo de yoga, o libro grande.
- Una silla.
- Una manta o frazadita para abrigar el cuerpo luego de la práctica.
